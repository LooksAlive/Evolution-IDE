Features

  • Text Editor

    Text editor consist of many functionalities. It is resizable, can transform text, has many code editable features

  • Debugger

    Integrated LLDB Debugger. Variable Views, Watch features, stepping, editing variables. All of this in friendly interface.

  • Education system

    Evolution has its own education system, which contains snippets for many C, C++ features to learn or use as an example in real life. User also can create his own snippets from his code or Add/Commit new code snippet to integrate into IDE.

  • Refactoring, searching

    Refactoring is the most required feature. Evolution offers sentensive friendly interface for this operations.

  • Cmake Support

    Automatic Cmake code is generated by default, offering sample code for various output transformation (Precompiled headers, Library management, Preprocessing, ...)

  • Formatter

    Based on clang-format tool we can geneate .clang-format file and use user custom format style (CLI invocation) or use built in format interface.

  • Linter

    Linter works upon a clang-tidy, clang-check API. Before compilation is invoked Linter is run at first to avoid this stage (if an error occured)
